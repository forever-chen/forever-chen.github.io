读写文件

写文件

读文件

  • readlines是读取所有内容，然后整体返回
  • readline琢行进行读并返回

    文件编码

  • 读写的时候编码格式应该一样，就不会出现乱码

    字符含义

  • w 写文本
  • r 读文本
  • a 在文本后面天际
  • wb 写二进制
  • rb 读二进制
  • ab 添加二进制
  • w+ 既可以读又可以写（创建文件）
  • r+ 读写，文件不存在报错
  • a+ 读写，在后面添加
  • x 创建

    文件目录管理

  • 主要用到os模块
  • os.getcwd()
  • os.listdir()
  • os.path.exists()
  • os.makedirs()
  • os.removedirs() //这个只有文件内容为空的时候可以用，否则就会报错
  • shutil
  • shutil rmtree() // 删除文件
  • os.path.isfile()
  • os.path.isdir()
  • os.exists()

    正则表达式

  • re.search 从每一行的开始进行匹配
  • re.match 从字符串的最开始进行匹配
  • re.findall
  • re.finditer
  • re.sub 返回替代后的字符串
  • re.subn 返回替代次数
  • 正则中加入一个()就会返回括号里面匹配的内容

    pickle和json

    对比 pickle json
    存储格式 python特定的bytes格式 通用json text格式，可用于常用的网络通信
    数据种类 类，功能，字典，列表，元组 基本和pickle一样，但不能存类，功能
    保存后可读性 不能直接阅读 能直接阅读
    跨语言性 只能用在python 可以跨多种语言
    处理时间 长（需要编码数据） 短（不需用编码数据）
    安全性 不安全 相对安全
